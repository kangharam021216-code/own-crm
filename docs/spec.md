# 화면/권한 요약

## 메뉴(직원)
- 대시보드
- 캘린더
- 예약
- 고객
- 보관함(MANAGER 이상)
- 직원/거래처 관리(ADMIN)
- 설정(ADMIN)

## 거래처(PARTNER)
- 거래처 캘린더(읽기 전용)
- 거래처 일정 리스트(읽기 전용)
- 본인에게 “할당된 일정/예약”만 보임
- 고객 정보: 이름 + Primary 연락처 1개
- 내부 메모/금액/로그/직원정보 비노출

## 삭제 정책
- 기본 삭제 = 보관(soft delete)
- 보관함에서 복구/완전삭제
- 완전삭제 권한 = MANAGER 이상

## 전시동 운영 규칙
- 웨딩 슬롯: 11:00 / 14:30 / 18:00
- 웨딩 사용시간: 3시간 블록(예식1 + 식사2)
- 대관: 시작~종료 자유
- 충돌: tsrange 겹치면 불가(대관은 겹치는 웨딩 슬롯만 막힘)
- 홀드: 기본 7일, 만료 시 자동 해제(문의 상태로)
